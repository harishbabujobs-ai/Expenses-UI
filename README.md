# Expense Tracker Frontend

React + Vite frontend for the Expense Tracker API.

## Features

- Login and register screens with polished UI
- JWT-based authentication
- Protected dashboard route
- Add and delete expenses from dashboard
- Responsive layout for desktop and mobile

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Set API URL (optional). By default, the app uses `http://localhost:5000/api`.

```bash
echo "VITE_API_URL=http://localhost:5000/api" > .env
```

3. Start development server:

```bash
npm run dev
```

## Build

```bash
npm run build
```
